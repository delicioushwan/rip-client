 내가 있는 현재의 위치를 똥으로 표시<br>
 주변의 화장실은 하트로 표시 <br>
 하트를 클릭하면 화장실의 상세주소와 별점 평가등을 확인 작성 <br>
 내가있는 위치에 화장실 추가기능 <br>
 좌측아래 아이콘을 터치하여 현재내위치로 지도중심 <br>
 지도를 이동하면 지도의 중심으로부터 화장실을 찾을수 있다.
